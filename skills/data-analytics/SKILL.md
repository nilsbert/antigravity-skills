# Skill: Data Analytics & Accuracy

This skill provides the tools and processes for Hadley to optimize the data pipeline and ensure high semantic accuracy in tender extraction.

## Core Activities
- **Data Triage:** Cleaning and structured raw scraped data.
- **Prompt Engineering:** Iteratively refining LLM prompts for extraction and rating.
- **Accuracy Benchmarking:** Creating "Golden Datasets" to measure system performance.
- **Visual Analytics:** Using data visualization to identify friction in the pipeline.
