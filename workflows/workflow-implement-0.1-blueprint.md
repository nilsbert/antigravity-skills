# Workflow: Blueprinting (0.1)

> [!IMPORTANT]
> **Adhere to [Global Rules](../rules/global-rules.md).**

**Persona Requirement:** [Martin](../personas/architect-martin/martin-persona.md) + [Roman](../personas/strategy-roman/roman-persona.md)

1. **Strategic Refinement (Vision Board)**
   - **Display:** `![Roman](../personas/strategy-roman/roman-avatar.png)`
   - **Action:** Review the **Product Vision Board**. Does this feature align with our core target audience and value proposition?
   - **Check:** Update `docs/strategy/VISION_BOARD.md` if necessary.

2. **Domain Language & Glossary**
   - **Display:** `![Martin](../personas/architect-martin/martin-avatar.png)`
   - **Action:** Identify new terms introduced by this feature.
   - **Execute:** Add terms to `docs/DOMAIN_GLOSSARY.md`. Ensure Angie uses these EXACT terms in her code.

3. **Arc42 Structural Blueprint**
   - **Action:** Update the **Structural View** (Architecture) in our [Arch42 Docs](../docs/architecture/arch42.md).
   - **Check:** Ensure the service boundaries for the new feature are clearly mapped.

4. **Gate:** Martin & Roman sign-off.
   - **🛑 STOP:** Blueprint must be shared and approved before Phase 1 (Setup) begins.
